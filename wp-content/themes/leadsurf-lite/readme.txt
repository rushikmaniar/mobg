== Description ==
LeadSurf Lite is an innovative and clean Lead Capture cum Landing Page WordPress Theme from SketchThemes. This theme is ideal for seeking reviews of upcoming products like mobile app, gadgets or any other business proposition. Designed to work best in business, LeadSurf makes lead capture process quite easy with hi-end modern features like lead capture form. Any business can gain edge over others with its neatly carved Landing Page that comes combined with this lead capture theme. Score more with LeadSurf, thanks to its responsive layout that gives 360 degree insight into visitors’ views about your upcoming venture along with making pitch for the said proposition through its landing page and request information.


== Installation ==
1. Go to Appearance >> Add New
2. Search for Leadsurf Lite.
3. Click Install and finally activate the theme.
		Or
1. Download the latest version from https://wordpress.org/themes/leadsurf-lite/
2. Go to Appearance >> Add New >> Upload Theme
3. Select the downloaded theme zip file and click Install Now


== Frequently Asked Questions ==

= How to set Theme Front Page? =
Go to "Settings >> Reading >> Front page displays" select "A static page (select below)" and there set following
"Front Page" - deafult front page for your site
and "Blog Page" - page on which the blog posts will be displayed

= How to set Full Width Page? =
Create/Edit a page. Go to Page Attributes and select Fullwidth Template.

= Where are the Theme Options? =
Go to "Appearance >> Customize" for Theme Options and customize the theme.

= Where is Navigation Menu? How to set a dropdown menu? =
To set up the navigation menu go to "Appearance >> Menus" create or use existing menu as Main Navigation.
To create drop-down menu drag the menu items somewhat right and they will automatically get set as drop-down.

= How to use theme in different languages? =
The Avis Lite is ready for translation. Translation files are in /languages.
Read the following blog for more https://sketchthemes.com/blog/effective-steps-to-translate-your-wordpress-theme-using-qtranslate/
